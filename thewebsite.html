Behind the scenes

I never created a website before,
so I got help
what a timesaver!

This website consists of some HTML, CSS, and hosting. Most of it was actually set up by Claude in an hour or so.


I once took a programming course at university, so I had just enough knowledge to know about the core elements of a website. But I also knew it would probably take me a very long time to build something like this on my own. My goal was simply to have a place where I could write what’s on my mind. I already had a rough idea of the design.


I didn’t actually start with Claude. There are so many tools out there now, and I had heard people were building and publishing websites through a platform called Lovable. So I started there, trying to explain my thoughts and design ideas. And honestly, you can absolutely build and host your dreams with that tool. But I wanted a bit more freedom and control. Also over my money, because this ain't free.


Then I tried Claude for the first time ever, and my mind was blown away. I watched it plan out the entire setup while also explaining the steps to me along the way. This completely removed the barriers to entry.


I learned how to configure a domain and set up the correct DNS records. Claude also showed me that I could use GitHub to run the entire website. I just needed to create a repository and add the files I wanted. Now I write posts in HTML or Markdown through Visual Studio Code, which is connected directly to my GitHub.


And that setup is more than enough for me.


That said… I think I if break something, it will probably be very very visible on the website...for a while... So please bear with me.
